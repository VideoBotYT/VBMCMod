In order to install this data pack, you need to unpack the zip archive with a program like 7Zip. 
You will get a folder called 'ija-planet-generator'.

Go into your Minecraft folder and in there, click on the folder called 'saves'.
This folder contains all your worlds. Open the folder for the world where you 
want to install this function. Then click on the folder 'datapacks' within
there, and then place the 'ija-planet-generator' folder inside.

Reload the world, and in there, you can type the following to create a planet:
/function ija-planet-generator:create-planet
